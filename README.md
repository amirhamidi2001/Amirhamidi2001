## 👋 Hey, I'm Amir

I'm a Full-Stack Web Developer based in **Tehran, Iran**, focused on building real-world products that actually work — not just demos.

My sweet spot is **backend architecture**: clean APIs, scalable Django services, and systems that hold up under pressure. On the frontend, I build with React and Tailwind to create interfaces that feel as good as the logic behind them.

I care about clean architecture, performance, and writing code that's easy to maintain six months later — including by myself.

> *Currently open to full-time roles, contract work, and interesting collaborations.*
